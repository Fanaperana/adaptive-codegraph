# Security Policy

## Supported Versions

adaptive-codegraph is early-stage software. Security fixes are provided for the
latest commit on the default branch.

| Version | Supported |
|---------|-----------|
| `main` / `master` | Yes |
| Older commits | No |

## Reporting a Vulnerability

Please do not report security vulnerabilities in public issues.

Use GitHub's private vulnerability reporting feature if it is available for this
repository. If it is not available, contact the repository owner through their
GitHub profile and include enough detail to reproduce and assess the issue.

Helpful details include:

- Affected command, crate, or MCP tool.
- Steps to reproduce.
- Impact and possible exploit scenario.
- Suggested mitigation, if known.

We will make a best effort to acknowledge reports promptly, investigate the
issue, and publish a fix or mitigation when appropriate.
